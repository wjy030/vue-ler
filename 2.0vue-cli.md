# vue2.0脚手架
## 2.0脚手架注册全局组件
```
import win from './win.vue'
const comps = {
  install: function (vue) {
    vue.component('win',win);
  }
}
Vue.use(comps);
new Vue({
  el: '#app',
  render: h => h(App)
})
```
* 创建组件注册实例,内有install方法,传入的vue对象
* 然后通过vue.component('win',win);将组件名和组件对象对应起来,
* 然后使用Vue.use(comps);注册这些组件.
* 被注册的组件能在其他任意组件模板中使用.
